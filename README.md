# Crypto Trading Bot

This is an automated crypto trading bot using machine learning (LightGBM) and Bitget's API.

## Features
- Live trading with Bitget
- ML-based buy/sell predictions
- Technical indicators (RSI, MACD, Bollinger Z-Score)
